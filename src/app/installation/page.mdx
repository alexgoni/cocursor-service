# 설치

## API Key

**CoCusror**를 사용하기 위해서는 API Key를 발급받아야 합니다.

// 로그인하고, API Key 발급받기

### 보안 모드

프로젝트를 배포하는 단계에서는 API Key 발급 시 **보안 모드** 사용을 권장합니다.  
보안 모드에서는 배포되는 URL을 추가로 입력하고, 서버에서 웹소켓 연결 시 해당 URL을 한 번 더 검증하여 안전한 연결을 보장합니다.

단, 보안 모드의 API Key를 사용하면 URL을 검증하므로, 개발 단계에서는 **CoCursor** 기능이 작동하지 않습니다.  
따라서, 개발 단계와 배포 단계에서 `env` 키를 두 가지로 관리하여 각각 사용하는 것을 권장합니다.

```env filename="env.local"
REACT_APP_COCURSOR_API_KEY=...
```

```env filename="env.production"
REACT_APP_COCURSOR_API_KEY=...
```


## Install Package

API Key를 발급받았다면, 아래 명령어를 실행하여 프로젝트에 CoCursor를 설치하세요.

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tabs.Tab>
  ```sh copy
  npm i cocursor
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```sh copy
  yarn add cocursor
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```sh copy
  pnpm add cocursor
  ```
  </Tabs.Tab>
</Tabs>

<br />

이후 **CoCursor**를 적용하고자 하는 페이지 컴포넌트 최상단에 다음과 같이 적용해주세요.

```tsx 
import CoCursorProvider from "cocursor";

export default function PageComponent() {
  return (
    <CoCursorProvider apiKey={process.env.REACT_APP_COCURSOR_API_KEY as string}>
      { /* ... */ }
    </CoCursorProvider>
  );
}
```
