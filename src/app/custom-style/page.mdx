# Custom Style

**CoCursor**에서 사용하는 모든 CSS입니다.  
해당하는 클래스 이름에 대해 명시도를 높여 원하는 스타일을 적용할 수 있습니다.

```css copy
.cocursor__cursor-wrapper {
  position: fixed;
  width: 14px;
  height: 14px;
  pointer-events: none;
  z-index: 2147483647;
}

.cocursor__arrow {
  width: 14px;
  height: 14px;
}

.cocursor__label {
  position: absolute;
  top: 8px;
  left: calc(100% + 4px);
  color: white;
  font-size: 12px;
  padding: 2px 6px;
  border-radius: 4px;
  white-space: nowrap;
}
```
